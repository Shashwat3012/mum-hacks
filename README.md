# mum-hacks
This is the Mumbai Hacks project repository.


## Setup

### node-api Setup
1. get into proper directory: `cd backend/node-api`
2. install dependencies: `npm i`
3. start backend node server: `npm run dev`


### FastAPI setup
1. get into proper directory: `cd backend/python-fastapi`
2. create venv using: `python -m venv venv`
3. enter the virtual env: `venv/Scripts/activate`
4. install dependencies: `pip install -r requirements.txt`
5. install other dependencies if required for your code, but be inside venv before installing
6. to run use this: `uvicorn main:app --reload --port 8001`
